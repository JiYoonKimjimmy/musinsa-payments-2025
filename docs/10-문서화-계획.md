## 4. 리소스 파일

### 4.1 다이어그램 위치

| 다이어그램      | 파일 위치                              |
|------------|------------------------------------|
| ERD        | `resource/erd.svg`, `.png`, `.pdf` |
| AWS 아키텍처   | `resource/aws-architecture.svg`    |
| Mermaid 소스 | `resource/scripts/*.mmd`           |

### 4.2 다이어그램 생성

```bash
# Mermaid CLI로 이미지 생성
npx @mermaid-js/mermaid-cli -i resource/scripts/erd.mmd -o resource/erd.svg
```

## 5. 버전 관리

### 5.1 Semantic Versioning

| 버전        | 변경 유형          |
|-----------|----------------|
| MAJOR.x.x | 호환되지 않는 API 변경 |
| x.MINOR.x | 백워드 호환 기능 추가   |
| x.x.PATCH | 백워드 호환 버그 수정   |

### 5.2 CHANGELOG 형식

| 섹션      | 내용       |
|---------|----------|
| Added   | 추가된 기능   |
| Changed | 변경된 기능   |
| Fixed   | 수정된 버그   |
| Removed | 제거된 기능   |
